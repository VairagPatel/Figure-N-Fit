# FigurenFit Frontend (React + Vite + Tailwind)

## Quickstart
```bash
npm i
npm run dev
```

When the backend is ready, create `.env.local` with:
```bash
VITE_API_BASE=http://localhost:8080/api
```

## Tech
- React 18 + TypeScript
- Vite 5
- Tailwind CSS 3
- React Router 6
- React Hook Form + Zod
- react-helmet-async
